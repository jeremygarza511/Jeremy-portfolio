# Jeremy Garza – Professional Portfolio

This project is a multi-page professional portfolio website developed as part of my coursework for D276: Web Development Foundations at Western Governors University.

The site showcases my resume, cover letter, and career goals while demonstrating my ability to build and structure a responsive website using HTML, CSS, and JavaScript.

##  Project Overview

This portfolio was designed to present both my technical skills and professional background in a clean, structured format. It reflects my transition into software engineering and IT by combining real-world experience with academic learning.

The project includes:

* Resume (professional experience and technical skills)
* Cover Letter (tailored for IT and software roles)
* Career Goals (short-term and long-term objectives)

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive design principles

## Key Features

* Multi-page navigation structure
* Consistent styling across all pages
* Custom layout using CSS grid and flexbox
* Interactive elements (navigation, forms, media)
* Clean, readable UI focused on professional presentation
